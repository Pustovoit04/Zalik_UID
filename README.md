# Книга рецептів

Мобільний застосунок для перегляду рецептів, створений на **React Native** з використанням **Expo**.

Проєкт виконано як залікову роботу з дисципліни **«Розробка інтерфейсів користувача»**.

---

## Опис проєкту

**Книга рецептів** — це мобільний застосунок, у якому рецепти згруповані за категоріями.

Користувач може обрати категорію, переглянути список рецептів у ній та відкрити детальну інформацію про конкретну страву: опис, час приготування, складність, інгредієнти та кроки приготування.

---

## Варіант завдання

**Варіант №22**

## Екрани застосунку

### Екран категорій

На головному екрані користувач бачить список категорій рецептів.  
Кожна категорія містить назву, короткий опис, іконку та кількість рецептів.

### Екран списку рецептів

Після вибору категорії відкривається список рецептів, які належать до неї.  
Кожна картка рецепта містить назву, опис, час приготування та складність.

### Екран деталей рецепта

На екрані деталей відображається повна інформація про рецепт:

- назва страви;
- короткий опис;
- час приготування;
- складність;
- список інгредієнтів;
- кроки приготування.

## Запуск проєкту

Клонувати репозиторій:

```bash
git clone https://github.com/Pustovoit04/Zalik_UID.git
```

Перейти в папку проєкту:

```bash
cd Zalik_UID
```

Встановити залежності:

```bash
npm install
```

Запустити застосунок:

```bash
npx expo start
```

Після запуску застосунок можна відкрити через **Expo Go**, **Android Emulator** або web-режим Expo.
