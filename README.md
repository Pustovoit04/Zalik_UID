# Книга рецептів

Мобільний додаток для перегляду рецептів, створений на React Native з використанням Expo.

## Опис проєкту

Додаток розробляється для залікової роботи з дисципліни «Розробка інтерфейсів користувача».

Тема варіанта: реалізувати додаток для перегляду рецептів з категоріями, списком рецептів у кожній категорії та екраном деталей з інгредієнтами і кроками приготування.

## Запланована функціональність

- перегляд категорій рецептів;
- перегляд списку рецептів у вибраній категорії;
- перехід на екран деталей рецепта;
- відображення інгредієнтів;
- відображення покрокового приготування.

## Поточний стан

На першому етапі створено базовий Expo-проєкт та стартовий екран додатку.

## Технології

- React Native
- Expo
- JavaScript
- Git / GitHub

## Запуск проєкту

1. Встановити залежності:

```bash
npm install
```

2. Запустити проєкт:

```bash
npm start
```

3. Відкрити додаток через Expo Go, Android Emulator або web-режим.
